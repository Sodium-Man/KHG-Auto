# KHG Auto Workshops Website

Official website for **KHG Auto Workshops**, a Clayton South based automotive business offering used cars, vehicle enquiries, sell/trade-in enquiries, finance enquiries, and workshop services.

Live website:  
https://khgautocarsales.com.au

---

## About the Website

This website is built as a static front-end website using HTML, CSS and JavaScript. It includes a responsive layout for desktop, laptop, tablet and mobile devices.

The website includes:

- Home page
- Vehicles for sale
- Dynamic vehicle detail pages
- Sell Your Car enquiry page
- Finance enquiry page
- Contact page
- Automotive services section
- Google review slider
- FormSubmit AJAX enquiry forms
- Responsive navigation menu
- Vehicle search, filters and pagination
